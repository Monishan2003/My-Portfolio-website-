Mathan Monishan - Portfolio Website
Overview
This is my professional portfolio website showcasing my skills, education, projects, and contact information. The website features a modern, responsive design with smooth animations and interactive elements to highlight my qualifications as a developer and IT student.

Features
Modern UI Design: Clean, professional interface with a cohesive color scheme

Responsive Layout: Fully responsive for all device sizes

Interactive Elements:

Animated typing text effect

Smooth scrolling navigation

Hover animations on cards and buttons

Education Timeline: Visual timeline showcasing academic journey

Project Showcase: Grid layout for displaying projects with GitHub links

Skills Section: Organized by category with icons

Contact Form: Functional form for direct communication

Technologies Used
Frontend:

HTML5, CSS3, JavaScript

Font Awesome icons

Libraries:

jQuery

Typed.js for typing animations

Design:

CSS Flexbox and Grid

CSS variables for consistent styling

Responsive design with media queries

Key Sections
Home: Hero section with animated typing text

About: Personal introduction and background

Education: Timeline format with academic history

Projects: Showcase of completed work with technologies used

Skills: Categorized technical abilities with icons

Contact: Contact information and message form

Footer: Quick links and social media

Setup and Usage
This is a static website requiring no special installation. Simply:

Clone the repository: git clone https://github.com/Monishan2003/portfolio.git

Open index.html in any modern browser

Navigate through the sections using the menu or scroll

Customization
To personalize this portfolio:

Update personal information in the HTML file

Replace the placeholder image with your own photo

Modify the education and project details

Adjust colors by changing CSS variables in the :root section

Add/remove skills as needed
